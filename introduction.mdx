# Eventos API

Eventos is a lightweight REST API that powers event participant management. Built with Node.js, Express, and PostgreSQL, it provides a simple HTTP interface for retrieving participant data stored in a PostgreSQL database.

## Overview

The Eventos API exposes a single endpoint that returns a list of all participants registered in the system. It is designed to serve a frontend application hosted at a known origin, with CORS enforced server-side.

| Technology | Purpose |
|---|---|
| Node.js 14.x | Runtime environment |
| Express 4 | HTTP server and routing |
| PostgreSQL | Relational database |
| `pg` | PostgreSQL client with connection pooling |
| `dotenv` | Environment variable loading |
| `cors` | Cross-Origin Resource Sharing middleware |

## How it works

The API connects to a PostgreSQL database using a connection pool configured from the `DATABASE_URL` environment variable. On each request to `GET /lista`, it runs a `SELECT *` query against the `Prueba` table and returns the results as a JSON array.

<Note>
  The API enforces CORS to only accept requests from `https://eventosutmach.000webhostapp.com`. Requests from other origins will be blocked by the browser.
</Note>
